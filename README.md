# Email Campaign – Support for Freedom and Democracy in Iran

## English

### Purpose of this project

This repository contains a simple web page that allows citizens to easily send a prepared email to members of the German Bundestag and relevant government offices.

The goal of this initiative is to raise awareness and encourage political support for the Iranian people's struggle for freedom, democracy, and human rights.

The page opens the user's email application with a prepared message addressed to multiple political representatives.

Due to technical limitations in email clients and browsers, the recipient list is split into smaller groups. Each click opens the next prepared email draft.

Users can review, edit, and send the message themselves.

### How it works

1. The website contains a button.
2. When clicked, the user's email client opens with:

   * a prepared subject
   * a prepared message
   * a list of recipients (BCC)
3. The user can review and send the email.

No data is collected.
No emails are sent automatically.

The user remains in full control.

### Email content summary

The message expresses support for:

* Freedom and democracy in Iran
* Respect for the territorial integrity of Iran
* Protection of civilians
* Political attention from European governments
* Support for a democratic transition

The email is written in German because the recipients are members of the German Bundestag.

### Disclaimer

This project is a civic engagement tool.
It does not represent any political organization or government.

The repository only provides a technical tool that allows individuals to contact their elected representatives.

Users are responsible for reviewing the email before sending it.

## Deutsch

### Zweck dieses Projekts

Dieses Repository enthält eine einfache Webseite, mit der Bürgerinnen und Bürger eine vorbereitete E-Mail an Mitglieder des Deutschen Bundestages sowie an relevante Regierungsstellen senden können.

Das Ziel dieser Initiative ist es, Aufmerksamkeit für den Freiheits- und Demokratiekampf des iranischen Volkes zu schaffen und politische Unterstützung dafür anzuregen.

Die Seite öffnet die E-Mail-App des Nutzers mit einer vorbereiteten Nachricht an mehrere politische Entscheidungsträger.

Aufgrund technischer Einschränkungen von Browsern und E-Mail-Programmen wird die Empfängerliste in mehrere kleinere Gruppen aufgeteilt. Jeder Klick öffnet eine weitere vorbereitete E-Mail.

Die Nutzer können die Nachricht vor dem Versand lesen, ändern oder ergänzen.

### Funktionsweise

1. Die Webseite enthält einen Button.
2. Beim Klicken öffnet sich die E-Mail-App mit:

   * einem vorbereiteten Betreff
   * einem vorbereiteten Nachrichtentext
   * einer Empfängerliste (BCC)
3. Der Nutzer kann die E-Mail prüfen und senden.

Es werden keine Daten gespeichert.
Es werden keine E-Mails automatisch versendet.

Die Nutzer behalten jederzeit die volle Kontrolle.

### Zusammenfassung des E-Mail-Inhalts

Die Nachricht spricht sich aus für:

* Freiheit und Demokratie im Iran
* die territoriale Integrität des Iran
* den Schutz der Zivilbevölkerung
* mehr politische Aufmerksamkeit in Europa
* Unterstützung eines demokratischen Übergangs

Die E-Mail ist auf Deutsch verfasst, da sie an Mitglieder des Deutschen Bundestages gerichtet ist.

### Hinweis

Dieses Projekt ist ein zivilgesellschaftliches Engagement-Werkzeug.
Es repräsentiert keine politische Organisation und keine staatliche Institution.

Das Repository stellt lediglich ein technisches Werkzeug bereit, mit dem Bürger ihre gewählten Vertreter kontaktieren können.

Die Nutzer sind dafür verantwortlich, den Inhalt der E-Mail vor dem Versand zu prüfen.
